Privacy Policy

Gabriel AI Assistant
Last updated: February 12, 2026

Overview

Gabriel is a personal AI assistant that accesses Google Workspace services (Gmail, Calendar, Drive, Contacts, Docs, Sheets) on behalf of its authorized user. This application is for private, single-user use only.

Data Collection

Gabriel accesses Google Workspace data solely to perform tasks requested by the authorized user. No data is collected, stored, shared, or sold to third parties.

Data Usage

Data accessed through Google APIs is used exclusively to:

  • Read and send emails on behalf of the user
  • View and manage calendar events
  • Access and organize files in Google Drive
  • Read and update Sheets and Docs

All data processing occurs in real-time and is not retained beyond the immediate task.

Data Sharing

No user data is shared with any third party. This application operates in a private, self-hosted environment.

Security

OAuth 2.0 credentials are stored securely on encrypted local storage. API access uses HTTPS exclusively.
